# CRUD API
This is a simple `Node.js, express.js mongodb` API to create data, such as Product, user etc.

## how to user it
1. **Clone this repository**

`git clone <repository>`

2. **Install node package**
`npm Install`

3. **Start the server**
`npm start`

### Customizaton
All source file in `src` folder

**File stuctures**


```javascript
src
  |_ all_route
    |_ route.js
  |_ db
    |_ db.js
  |_ app.js  
```
